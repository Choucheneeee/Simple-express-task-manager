# Simple Express Task Manager

A simple Task Manager API built with Node.js and Express. It allows you to create, read, update, and delete tasks (CRUD operations) stored in a JSON file.

---

## Features

- **Get All Tasks**: Retrieve a list of all tasks.
- **Get a Task by ID**: Retrieve a specific task by its ID.
- **Create a Task**: Add a new task with a title, description, and status.
- **Update a Task**: Modify an existing task's details.
- **Delete a Task**: Remove a specific task by ID.
- **Delete All Tasks**: Clear the entire task list.
